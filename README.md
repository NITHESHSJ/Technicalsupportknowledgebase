# 📘 Technical Support Knowledge Base (TSKB)

## 📖 Description
The **Technical Support Knowledge Base (TSKB)** is a web-based application designed to help users quickly find solutions to common technical problems. It provides structured content such as FAQs, troubleshooting guides, and tutorials, enabling users to resolve issues independently without needing direct support.

---

## 🚀 Features
- 🔍 Search functionality for quick issue resolution  
- 📂 Categorized knowledge articles  
- 👤 Role-based access control (Admin/User)  
- 📝 Add, update, and delete articles (Admin)  
- 📊 User-friendly and responsive UI  
- ⚡ Fast and efficient backend processing  

---

## 🛠️ Tech Stack
**React.js, Node.js, Express.js, MongoDB, HTML, CSS, JavaScript, Tailwind CSS**

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
```bash
git clone https://github.com/your-username/tskb.git
cd tskb
```

### 2️⃣ Install dependencies
```bash
# Install backend dependencies
npm install

# Install frontend dependencies
npm install --prefix frontend
```

### 3️⃣ Run the application
```bash
# Start backend server
npm start

# Start frontend
npm start --prefix frontend
```
